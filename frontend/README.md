# Shed Frontend Application 

## Prerequisites

- [pnpm](https://pnpm.io/)
- Node.js version 20.x or above 

## Getting Started

- Run `pnpm install` from this directory
- To start the server, run `pnpm dev` from this directory, and the app should start on http://localhost:5173
